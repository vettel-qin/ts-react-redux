### 工具版本

- node 版本 `v10.16.0`
- npm 版本 `v6.12.0`

### 开发命令

1. 运行 `npm install` 安装依赖包
2. 运行 `npm run start` 启动 dev 服务器
3. 运行 `npm run build` 编译发布包
